# Aegis Frontend

The sleek, modern Next.js landing page and storefront for the Aegis OKX Agent platform. It serves as the primary gateway for users to understand Aegis's capabilities and acts as a bridge to our Agent Service Provider (ASP) listing on the OKX Agent Store.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4
- **UI Components**: Shadcn, Lucide React, Base UI
- **Language**: TypeScript

## Key Sections
- **Hero**: Introduces the Aegis platform with a striking headline and direct call to action.
- **Features**: Highlights core capabilities (Self-Healing Compiler, Secure Code Gen, Unsigned Tx Payloads).
- **Marketplace Interface**: A visual representation of our A2A and A2MCP services available on the OKX AI network.
- **Agent Integration**: Direct links to interact with the Aegis OKX Agent.

## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```
   *(Note: This project uses `pnpm` configuration overrides for specific packages, but `npm install` works smoothly.)*

2. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:3000`.

3. **Build for Production:**
   ```bash
   npm run build
   npm start
   ```
